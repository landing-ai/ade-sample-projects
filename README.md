# ade-sample-projects
Sample workflows, sample schemas, sample documents and sample front-ends using Agentic Document Extraction.

## Getting Started

The notebooks in this repository are based on the [ADE Quickstart Guide](https://docs.landing.ai/ade/ade-quickstart).

## Setup

1. Create a virtual environment:
```bash
python3.12 -m venv .venv
source .venv/bin/activate
```

2. Install required packages:
```bash
pip install ipykernel jupyter landingai_ade
```

3. Register the kernel:
```bash
python -m ipykernel install --user --name=ade-sample-projects
```

4. Open the notebooks and select the `ade-sample-projects` kernel.
